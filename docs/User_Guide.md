# User Guide

## First run

1. Download the repository or a release package.
2. Run `tools/Run_Anomaly_Safety_System_From_Source.bat`, or run the official EXE release if available.
3. Select your Anomaly/Zona/MO2 paths when the tool asks for setup information.
4. Choose a Tool Storage Folder where backups and profiles should be stored.

## Recommended use

Before heavy tweaking:

- Create a safety backup / safepoint.
- Back up your current MO2 profile.
- Back up your current graphics settings.
- Clear shader cache only when you are ready to let the game regenerate shaders.

## Important

Do not store your safety backups inside folders you plan to delete, overwrite, or use as temporary test folders.
