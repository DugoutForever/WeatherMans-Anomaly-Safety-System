# Security Policy

## Safety and transparency

WeatherMans Anomaly Safety System is provided with source files so users, moderators, and community members can inspect what the tool does.

## Antivirus false positives

Some antivirus tools may flag small unsigned utilities, especially if they are built from PowerShell or interact with files, folders, backups, archives, or game paths.

This repository includes source files for transparency.

## What the tool should never do

This tool should never:

- Steal user data
- Contact unknown servers
- Modify unrelated personal files
- Delete unrelated game files
- Bundle third-party mods or game assets
- Hide what it is doing from the user

If you find unsafe behavior, please report it through GitHub Issues.
